# MECHANISM-AI Architecture

MECHANISM-AI links biological modeling across scales:

- Molecular: binding and structural features
- Receptor: activation, clustering, dimerization
- Signaling: pathway dynamics such as NF-kB activation
- Cellular: cytokine output, phenotype state
- Tissue: future extension for spatial and multicellular behavior

## Core layers

1. Mechanistic models
2. Hybrid models
3. Interfaces
4. Cross-scale mappings
5. Uncertainty quantification
6. Mediator/orchestration
7. Closed-loop workflow

## Design goal

To support iterative AI-guided scientific discovery by linking structure, dynamics, uncertainty, and experimental feedback.
