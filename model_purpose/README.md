# Model Purpose

This directory classifies models by their functional role in the MECHANISM-AI workflow.

- `forward/`: models mapping inputs to system-level outputs
- `inverse/`: models identifying inputs likely to achieve desired outcomes
- `surrogate/`: fast approximations of complex models
- `hypothesis/`: alternative mechanistic or conceptual models used to test assumptions

This layer is organizational and may include references, wrappers, and metadata rather than full standalone implementations.
